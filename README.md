# UGA Templates

A collection of University of Georgia (UGA) academic templates I designed and use
for my own research, shared publicly for anyone who finds them useful.

## Contents

| Folder | What's inside |
| --- | --- |
| [`poster_ppt/`](poster_ppt/) | Conference poster template (PowerPoint, 48 × 36 in) + PDF preview |
| [`slides_ppt/`](slides_ppt/) | Presentation / dissertation-defense slide deck (PowerPoint, 16:9) |
| [`dissertation_overleaf/`](dissertation_overleaf/) | Dissertation LaTeX (Overleaf) template — _coming soon_ |

## Usage

Download the file for the template you want, then edit it in PowerPoint (or Overleaf
for the dissertation template). Feel free to adapt the design to your own work.
